# Product Context

## Why this exists
The project aims to demonstrate high-performance, browser-based RTS gameplay using the Rust/Bevy ecosystem. It fills a niche for Lovecraftian-themed strategy games with modern engine features.

## How it works
- **Engine**: Bevy 0.17 (ECS-driven).
- **Physics**: Avian3d for collision and movement.
- **AI**: Big-brain Utility AI for unit decision making.
- **Input**: Leafwing-input-manager for flexible controls.
- **Selection**: Built-in Bevy picking system.
- **UI**: Bevy_egui for tool and game interfaces.

## User Experience
Players control a cult leader and their followers, exploring a procedurally generated map, managing resources (energy/materials), and engaging in combat with other cults or corrupted creatures.
